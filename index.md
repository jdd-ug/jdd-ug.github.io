# JDDUG; Japan Datadog User Group

JDDUGとは、Datadogクラウドサービスを利用するユーザーのコミュニティです。

![JDDUG meetup](/assets/images/top_meetup.png)

## 開催予定

JDDUGは、少なくとも3ヶ月に1回のペースでユーザー会を開催する計画です。
次回は8月7日(水)19:00を計画中です。近日、イベント募集を公開します。
